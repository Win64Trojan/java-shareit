# Описание проекта

Проект java-shareit — это многомодульное веб‑приложение на Java, реализующее систему аренды вещей между пользователями. Приложение разделено на два основных модуля: gateway и server, что позволяет использовать архитектуру микросервисов.

## Основные функции

* Аренда вещей: Пользователи могут создавать объявления о вещах, которые хотят сдать в аренду.
* Запросы на аренду: Другие пользователи могут отправлять запросы на бронирование вещей.
* Управление запросами: Владелец вещи может одобрять или отклонять запросы.
* Отзывы и рейтинги: Возможность оставлять отзывы после завершения аренды.

## Технологии

* Java 21 — основной язык программирования.
* Spring Boot 3.3.2 — фреймворк для разработки веб‑приложений.
* Maven — система сборки и управления зависимостями.
* PostgreSQL 16.1 — СУБД для хранения данных.
* Docker — контейнеризация сервисов.
* Docker Compose — управление многоконтейнерными приложениями.
* Lombok — библиотека для упрощения кода.
* JUnit 5 — фреймворк для тестирования.
* Checkstyle — статический анализ кода.

## Структура проекта

### Модули

* **gateway** — API‑шлюз, обеспечивающий маршрутизацию запросов между клиентом и сервером.
* **server** — основной модуль сервера, реализующий бизнес‑логику и взаимодействие с базой данных.

### Docker Compose

Проект включает docker‑compose.yml, который позволяет запустить все компоненты приложения:

* **gateway** — сервис для обработки запросов.
* **server** — основной сервис с логикой приложения.
* **db** — PostgreSQL база данных.
